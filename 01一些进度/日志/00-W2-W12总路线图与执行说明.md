## 4. 范围判断

> 做出一个浏览器内、本地优先、加密存储、算得对、能导入导出、有图表、有性能数据、能支撑论文 System Design / Implementation / Evaluation 三章的个人交易账本原型。

- IndexedDB 持久化。
- 导入导出。
- 加密。
- benchmark。
- 论文素材。

P1 可压缩：

- 第二、第三个图表。
- 100k 量级 benchmark。
- UI 细节美化。

P2 可整块砍：

- NLP。
- Agent。
- 实时价格 API。
- K 线、技术指标、策略回测。
- 多端同步。
- Tauri / Capacitor 打包。
