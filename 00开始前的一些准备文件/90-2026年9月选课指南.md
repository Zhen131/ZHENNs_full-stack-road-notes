# 2026年9月选课指南

> 目标：为下学期选课提前做决策。等 2026 年 8 月开放选课后，按这份页面检查并正式选择课程。

## 一、当前选课目标

这次选课不是单纯为了补学分，而是为了把本科最后阶段逐步推向：

- AI Agent 开发
- Web 应用开发
- AI + 投资工具
- AI + 自媒体工具
- Web3 / 区块链方向的后续扩展

因此优先选择和 **AI、NLP、机器学习、Web 开发、安全、实习实践** 有关的课程。

## 五、课程价值判断

### 高优先级

#### Web Application Development

必须完成，而且和之后做 Web 工具、AI Agent 前端、投资账本系统都有直接关系。

#### Introduction to Natural Language Processing

非常建议选。

原因：

- 和 AI Agent 开发高度相关
- 可以做项目通过，适合沉淀作品集
- 可以尝试做 RAG、文档助手、文本分类、摘要、信息抽取、对话系统等方向
- 能和 Web Application Development 结合，做成一个完整小项目

#### Professional Training

非常关键，12 学分。如果通过，能大幅推进选修学分进度。

### 中高优先级

#### Introduction to Generative AI

和当下 AI 工具开发、内容生产、Agent 应用都很相关，建议保留。

#### Machine Learning in Practice

偏实践，适合把机器学习从理论推进到可展示项目。

### 可调整课程

#### Introduction to Reinforcement Learning

方向不错，但如果 42 学分压力太大，可以考虑放到最后一个学期。

RL 对 AI Agent 有一定关系，但本科阶段它通常比 NLP、Generative AI、ML Practice 更抽象，短期对作品集和就业项目的直接收益可能稍弱。

## 六、暂不优先考虑的课程

### Operation of infocommunication systems

这门课是 6 学分，补学分效率高，但目前判断不建议优先选。

原因：

- 之前听过几次，感觉难度偏高
- 内容可能偏 Cisco、网络系统、通信系统、设备配置
- 对 AI Agent、Web 应用、投资工具方向的直接帮助不如 NLP / Generative AI / ML Practice
- 如果不是为了网络工程、运维、安全基础设施方向，不建议为了 6 学分硬选

## 七、推荐选课方案

### 方案 A：完整 AI 主题学期

| 课程 | 学分 |
| --- | ---: |
| Foundations of Artificial Intelligence | 6 |
| Foundations of Computer Security | 6 |
| Web Application Development | 6 |
| Professional Training | 12 |
| Introduction to Natural Language Processing | 3 |
| Introduction to Generative AI | 3 |
| Introduction to Reinforcement Learning | 3 |
| Machine Learning in Practice | 3 |
| **总计** | **42** |

优点：

- AI 方向非常完整
- 学分推进快
- 和未来方向高度一致

风险：

- 学期负担很重
- 需要同时处理考试、项目、实习和多个 AI 课程

### 方案 B：更稳的版本

| 课程 | 学分 |
| --- | ---: |
| Foundations of Artificial Intelligence | 6 |
| Foundations of Computer Security | 6 |
| Web Application Development | 6 |
| Professional Training | 12 |
| Introduction to Natural Language Processing | 3 |
| Introduction to Generative AI | 3 |
| Machine Learning in Practice | 3 |
| **总计** | **39** |

这个方案把 **Introduction to Reinforcement Learning** 放到最后一个学期。

优点：

- 仍然保持 AI 主线
- 压力比 42 学分小一点
- 更容易把 NLP / Web App / ML Practice 做成真正可展示的项目

目前最推荐的方向是：

> Web Application Development + Professional Training + NLP + Generative AI + Machine Learning in Practice
